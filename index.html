Juego 00

Juego con el cual practique JAVA en la universidad, es un proyecto que usa puramente las herramientas brindadas por el paquete SWING de Java. El juego consiste en "eliminar" las gemas de colores evitando varios disparos que hacen diferentes enemigos, en total contaba con 7 niveles.

Juego 01

Al igual que el ejemplo anterior, este juego se hizo en su mayoría con las librerías nativas de JAVA salvo que esta vez se uso un JAR de terceros para la parte de la musicalización. El juego era un beat'em up con 6 niveles y sus respectivos jefes finales; contaba también con créditos finales y una tabla de score.

REST API PHP con JWT

Plantilla base para el desarrollo de una API en PHP con comprobación de un token web usando la librería firebase para el lenguaje antes mencionado. Se puede encontrar el ejemplo del código en mi repositorio.
